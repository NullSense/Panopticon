# Suggested Commands

## Build & Run
- `cargo build` — Build the project
- `cargo run` — Run the TUI dashboard
- `cargo run -- --init` — Initialize configuration
- `cargo run -- internal-hook --event start` — Test hook handler

## Testing
- `cargo test` — Run all tests
- `cargo test <test_name>` — Run a specific test
- `cargo test --test claude_setup` — Run Claude setup tests

## Code Quality
- `cargo clippy` — Run linter
- `cargo fmt` — Format code
- `cargo fmt -- --check` — Check formatting without modifying

## System Utils (macOS/Darwin)
- `git` — Version control
- `ls` — List files
- `open` — Open files/URLs (macOS-specific)
